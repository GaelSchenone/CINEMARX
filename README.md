# *CINEMARX*
CINEMARX es una app hecha con Java y base de datos MariaDB que simula una aplicación de compra de boletos, comida y asientos de un cine. El programa intenta simular, inspirandose grandemente en CineMark, la app de un cine por la cual comprar tus boletos y bebida para evitar esos trámites precencialmente en el establecimiento

## Funcionalidades:
En la App, usuarios puedes registrarse e iniciar sesión. La app cuenta con una UI llamativa desde la cual podes seleccionar tu función de forma visual, con carteleras y sinopsis de las películas, lo mismo para los múltiples combos de la barra del cine.
Cada sala y función cuenta con un apartado visual de selección de asientos que muestra la disponibilidad de los mismos para cada función. Se dispone de múltiples métodos para realizar pagos y generador de comprobantes, con QR y descargables

## Programación:
Cinemarx se hizo con Java en NetBeans, utilizando JSwing y divido en módulos, los cuales se podian ver dentro del github del proyecto donde cada módulo podía pushear sus cambios y bajar los de los demás y facilitar el trabajo simultaneo del equipo en distintas partes del código
La base de datos fue programada en su totalidad con MariaDB en DBeaver y fue hosteada gracias a Aguilucho Network.
